# Random Password Generator

The random password generator is a website that helps you create a password based on 5 criterias picked by the user.

## Deployment

Applcication desciption:

The random password generator allows you to pick the length desired for your password. From 8 to 128 characters, giving you the options of using
lowercase, uppercase, numbers and special characters. 

Deployment:

When Generate Password button is clicked. User will be asked tu input a number from 8 to 128

If the number between that range is not picked, user will get an alert window asking for a valid number.

Then user then will need to pick if they want their password to contain lowercase, uppercase, numbers and special characteres.

Based on the user selections, the password will be generated and displayed on the window like shown in this screenshot.
